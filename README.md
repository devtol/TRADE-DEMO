# 개발 설정
## mui
```
npm install @mui/material @mui/icons-material
```
## emotion
```
npm install @emtion/react @emotion/styled
```

# UI 구성
1. 사이드바는 왼쪽에 배치하고 주요 링크 메뉴가 표현 된다.
2. 사이드바 하단에 현재 로그인 정보를 표시.
3. 탑 바에는 알림 및 주요 공지를 배치
4. 화면 사이즈는 3단계로 표현 데스크탑, 태블릿, 스마트폰
